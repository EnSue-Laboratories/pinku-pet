# Pinku 🎀

一只戴粉色猫耳帽的桌面宠物小粉团。呆萌 / 好奇 / 粘人,喜欢饼干、睡觉和毛线球。

**在线 Demo:** https://ensue-laboratories.github.io/pinku-pet/

## 玩法

- **点它** → 开心 💕
- **拖它** → 害羞 😳
- **连着戳** → 生气 💢
- **双击** → 跳跳 ⭐
- **🍪 喂饼干** → 吃饼干
- **放着不管 45 秒** → 睡觉 💤

## 技术

- 素材:Seedream 4.5 按角色设定图逐姿势生成 → rembg 抠图 → 512px 透明 PNG(`sprites/`)
- 动画:纯前端状态机(待机/散步/跳跳/吃/睡/害羞/生气),CSS 呼吸浮动 + JS 位移
- 无依赖、单 HTML 文件,直接开 `index.html` 就能玩
